## System Language

OsmAnd Web now adapts to the language settings of your browser. By default, the system will use the language set in your browser preferences:

- **Chrome users**: The priority is set through `chrome://settings/languages`.

If you'd like to manually configure the interface (system) language:

- Go to *Menu ‚Üí* ‚öô *‚Üí Turn off language* auto-detection to choose your preferred language.


## New Map Context menu

In this update, we've introduced a new context menu for the map, making it easier to interact with points on the map. To open the menu, simply right-click anywhere on the map. The following actions are available:

- **Where am I** ‚Äì Quickly find your current location on the map
- **Open weather** ‚Äì Open the Weather menu.
- **Create new route** ‚Äì Plan a route from the selected location.
- **Add Favorite** ‚Äì Mark a point of interest as a favorite for easy access.
- **Directions from** ‚Äì Start navigation from the selected point.
- **Directions to** ‚Äì Get directions to the selected point.
- **Cope coordinates** ‚Äì Copy the coordinates for external use.
- **Add pin** ‚Äì Place a pin to mark a location on the map.
- **Show regions** ‚Äì Display the regions associated with the selected point.

![Map Context menu](./img/map_context_menu.png)

## Explore map menu

Our **Explore Map** feature has received a significant upgrade, making it easier to search for places and view points of interest (POIs) with their photos ([wikidata source](https://www.wikidata.org/)) directly on the map.  

To get started:

1. Click the üîç  icon to open the search tab, which displays popular POI categories and the "Explore" menu below. 
2. The "Explore" data will automatically be shown on the map.
3. Select "Show all" in the "Explore" section to open the full list with the _Filter_ button at the top.

![Explore menu](./img/explore.png)

4. Pressing the _Filter_ button opens the Categories menu for "Explore." To refine your search, open the Categories menu and select the items you are interested in.

![Explore menu](./img/explore_cat.png)

### POI Context Menu

Clicking on a POI now brings up an updated context menu that includes:

- POI Name (Category).
- Direction and distance to the POI.
- Detailed description.
- Access to the [Online photo gallery](#photo-gallery) for that POI.

![Explore menu](./img/poi_context.png)

## Additional Updates and Fixes

Along with these new features, we've addressed several important issues:

- **Banner for OsmAnd mobile**. A new [banner](https://github.com/osmandapp/OsmAnd/issues/18825) provides information about the OsmAnd mobile app, including a direct download link (coming in version 4.8).

- **Precipitation data fix**. We corrected the display of [precipitation data](https://github.com/osmandapp/OsmAnd-Issues/issues/2578) to ensure accuracy.

- **User Data Lag Fix**. Improved performance when handling [huge amount of user data](https://github.com/osmandapp/web/issues/384), reducing lag.

- **Zooming Fix**. Resolved issues related to [zooming](https://github.com/osmandapp/web/issues/246) when enabling favorites or tracks.
